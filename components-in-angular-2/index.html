Components in Angular 2

06 March 2017

What is a component ?

We can think of Angular 2 applications as a set of components similar to building blocks. Basically, they control the view rendered to the user and can be reused many times within an application.

An angular component includes :

  • Template
  • Class
  • Metadata
import { Component } from '@angular/core';

@Component({
selector: 'app-root',  
templateUrl: './app.component.html',  
styleUrls: ['./app.component.css']  
})

export class AppComponent {  
pageTitle: string = 'ET.techlab';  
}
Template

'./app.component.html' template is a view layout created with html. Includes bindings and directives

Class

Class is for supporting view created with TypeScript and it has :

  • Properties
  • Methods

export keyword makes the AppComponent available for use by other components and since the file containing class exports something now it is in ES module and will be loaded by module loader no need to add script tag for it. The pageTitle is a property which defines data element associated with a class and it's data type is string

Metadata

A class becomes angular component when we give to it a metadata. Angular needs that metadata to how to instantiate component, construct the view and interact with the component. Basically, decorators is a function that adds metadata to a class, it's members or method arguments prefixed with @ sign. For ex; @Component() is a built-in angular decorator which identifies class as a component. selector: 'app-root' selector is used to reference component in any other html by directive name, directive is simply custom html tag.

Before we use external class or function we have to tell the module loader where to get it like : import { Component } from '@angular/core';

import allows to use exported members from external ES modules from third party library, our own modules or from angular. We can import from angular because it's modular or collection of module libraries. Each library is itself a module related with several feature modules.

  • @angular/core
  • @angular/animate
  • @angular/http
  • @angular/router
